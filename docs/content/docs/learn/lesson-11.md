title: "11. Returning Answers"
description: "A function can also give back an answer with the keyword 'return'."
---

A function can also give back an answer with the keyword 'return'. Call the function, then use its answer. Try making your own math function!

## Code

```aksa
function area(side) {
    return side * side
}
print("Area of a 5 square:", area(5))
print("Area of a 10 square:", area(10))
```

Copy this code into the Aksa editor, run it, then change something and run it again!
