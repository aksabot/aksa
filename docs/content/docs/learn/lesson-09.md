---
title: "9. True & False"
description: "Something can only be true or false — nothing in between."
---

Something can only be true or false — nothing in between. You can store it in a variable and use it to choose. Join them with the keywords 'and', 'or', and 'not'. Try changing 'true' to 'false'!

## Code

```aksa
make had_breakfast = true
make is_raining = false
if (had_breakfast and not is_raining) {
    print("Let's play outside!")
} else {
    print("Let's play indoors instead")
}
```

Copy this code into the Aksa editor, run it, then change something and run it again!
