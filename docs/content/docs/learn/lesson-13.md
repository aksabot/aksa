description: "The robot draws while it walks."
---

The robot draws while it walks. 'forward' moves it, 'turn_right' spins it. Four forwards and turns make a square. Try changing the numbers!

## Code

```aksa
color("blue")
repeat 4 {
    forward(100)
    turn_right(90)
}
```

Copy this code into the Aksa editor, run it, then change something and run it again!
