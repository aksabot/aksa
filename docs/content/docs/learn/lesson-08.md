'while' keeps repeating as long as its condition is true. This program counts down like a rocket. Careful: change the number inside, or it will never stop!

## Code

```aksa
make number = 5
while (number > 0) {
    print(number)
    number = number - 1
}
print("Liftoff! 🚀")
```

Copy this code into the Aksa editor, run it, then change something and run it again!
