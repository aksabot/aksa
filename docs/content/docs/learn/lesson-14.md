Project time! If every step is a little longer than the last, you get a spiral. Change the color, numbers, and turns — make a spiral that is truly yours!

## Code

```aksa
make step = 5
color("purple")
repeat 30 {
    forward(step)
    turn_right(25)
    step = step + 3
}
```

Copy this code into the Aksa editor, run it, then change something and run it again!
