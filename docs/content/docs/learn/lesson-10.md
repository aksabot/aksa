A function is a command you invent yourself. Write the recipe once with the keyword 'function', then call its name any time. Try making your own greeting!

## Code

```aksa
function greet(name) {
    print("Hello,", name, "!")
}
greet("Aksa")
greet("Bima")
greet("Mom")
```

Copy this code into the Aksa editor, run it, then change something and run it again!
