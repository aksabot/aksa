---
title: "6. Memilih"
description: "Dengan 'jika', program bisa memilih."
---

Dengan 'jika', program bisa memilih. Kalau bagian dalam ( ) benar, jalankan blok pertama; kalau tidak, blok 'lainnya'. Coba jawab dengan angka berbeda!

## Kode

```aksa
umur = tanya("Berapa umurmu?")
jika (umur >= 10) {
    cetak("Wah, kamu sudah besar!")
} lainnya {
    cetak("Kamu masih kecil, tapi hebat!")
}
```

Salin kode ini ke editor Aksa, jalankan, lalu ubah sesuatu dan jalankan lagi!
