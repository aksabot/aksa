---
title: "3. Berhitung"
description: "Komputer jago berhitung."
---

Komputer jago berhitung. Pakai + untuk tambah, - untuk kurang, * untuk kali, dan / untuk bagi. Coba ganti angkanya dan lihat hasilnya berubah!

## Kode

```aksa
buat apel = 4
buat jeruk = 3
tulis("Jumlah buah:", apel + jeruk)
tulis("Dua kali apel:", apel * 2)
tulis("Sisa jeruk:", jeruk - 1)
```

Salin kode ini ke editor Aksa, jalankan, lalu ubah sesuatu dan jalankan lagi!
