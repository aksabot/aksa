Variabel itu seperti kotak berlabel untuk menyimpan sesuatu. Buat kotaknya dengan kata kunci 'buat', lalu pakai namanya di mana saja. Coba ganti isinya!

## Kode

```aksa
buat nama = "Aksa"
buat umur = 8
tulis(nama, "berumur", umur, "tahun")
```

Salin kode ini ke editor Aksa, jalankan, lalu ubah sesuatu dan jalankan lagi!
