---
title: "1. Menulis"
description: "Komputer bisa menulis apa saja untukmu."
---

Komputer bisa menulis apa saja untukmu. Perintah 'tulis' menampilkan teks di layar. Coba jalankan, lalu ubah teksnya menjadi kata-katamu sendiri!

## Kode

```aksa
tulis("Halo, dunia!")
tulis("Aku sedang belajar Aksa")
```

Salin kode ini ke editor Aksa, jalankan, lalu ubah sesuatu dan jalankan lagi!
