---
title: "7. Menghitung"
description: "Mau tahu sedang di putaran ke berapa?"
---

Mau tahu sedang di putaran ke berapa? Buat sebuah penghitung, lalu tambah 1 di setiap putaran. Inilah cara membuat pengulangan berangka (seperti "for")!

## Kode

```aksa
buat i = 1
ulangi 5 {
    tulis("Langkah ke", i)
    i = i + 1
}
```

Salin kode ini ke editor Aksa, jalankan, lalu ubah sesuatu dan jalankan lagi!
