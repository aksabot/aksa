---
title: "5. Mengulang"
description: "Malas menulis perintah yang sama berkali-kali?"
---

Malas menulis perintah yang sama berkali-kali? Pakai 'ulangi'! Semua yang ada di dalam { } akan diulang. Coba ubah angkanya menjadi 10.

## Kode

```aksa
ulangi 3 {
    tulis("Aksa itu seru!")
}
```

Salin kode ini ke editor Aksa, jalankan, lalu ubah sesuatu dan jalankan lagi!
