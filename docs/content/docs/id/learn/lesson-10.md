title: "10. Fungsi"
description: "Fungsi adalah perintah buatanmu sendiri."
---

Fungsi adalah perintah buatanmu sendiri. Tulis resepnya sekali dengan kata kunci 'fungsi', lalu panggil namanya kapan saja. Coba buat fungsi sapaanmu sendiri!

## Kode

```aksa
fungsi sapa(nama) {
    tulis("Halo,", nama, "!")
}
sapa("Aksa")
sapa("Bima")
sapa("Ibu")
```

Salin kode ini ke editor Aksa, jalankan, lalu ubah sesuatu dan jalankan lagi!
