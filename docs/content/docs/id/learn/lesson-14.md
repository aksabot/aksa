Saatnya proyek! Kalau setiap langkah sedikit lebih panjang dari sebelumnya, jadilah spiral. Ubah warna, angka, dan belokannya — buat spiral khasmu sendiri!

## Kode

```aksa
buat langkah = 5
warna("ungu")
ulangi 30 {
    maju(langkah)
    belok_kanan(25)
    langkah = langkah + 3
}
```

Salin kode ini ke editor Aksa, jalankan, lalu ubah sesuatu dan jalankan lagi!
