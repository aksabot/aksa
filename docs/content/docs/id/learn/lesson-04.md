---
title: "4. Bertanya"
description: "Program bisa bertanya kepadamu dengan perintah 'tanya'."
---

Program bisa bertanya kepadamu dengan perintah 'tanya'. Jawabanmu tersimpan di variabel. Jalankan, lalu ketik jawabanmu di kotak hitam bawah.

## Kode

```aksa
nama = tanya("Siapa namamu?")
cetak("Halo,", nama, "! Senang bertemu denganmu")
```

Salin kode ini ke editor Aksa, jalankan, lalu ubah sesuatu dan jalankan lagi!
