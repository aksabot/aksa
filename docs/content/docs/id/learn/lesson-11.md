---

Fungsi juga bisa mengembalikan jawaban dengan kata kunci 'kembali'. Panggil fungsinya, lalu pakai jawabannya. Coba buat fungsi hitungmu sendiri!

## Kode

```aksa
fungsi luas(sisi) {
    kembali sisi * sisi
}
cetak("Luas kotak sisi 5:", luas(5))
cetak("Luas kotak sisi 10:", luas(10))
```

Salin kode ini ke editor Aksa, jalankan, lalu ubah sesuatu dan jalankan lagi!
