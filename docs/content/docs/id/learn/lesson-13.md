Robot menggambar sambil berjalan. 'maju' menggerakkannya, 'belok_kanan' memutarnya. Empat kali maju dan belok membuat kotak. Coba ubah angkanya!

## Kode

```aksa
warna("biru")
ulangi 4 {
    maju(100)
    belok_kanan(90)
}
```

Salin kode ini ke editor Aksa, jalankan, lalu ubah sesuatu dan jalankan lagi!
