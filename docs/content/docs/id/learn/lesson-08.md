'selama' mengulang terus selama syaratnya masih benar. Program ini menghitung mundur seperti roket. Awas, jangan lupa mengubah angkanya, atau tidak akan pernah berhenti!

## Kode

```aksa
buat angka = 5
selama (angka > 0) {
    tulis(angka)
    angka = angka - 1
}
tulis("Meluncur! 🚀")
```

Salin kode ini ke editor Aksa, jalankan, lalu ubah sesuatu dan jalankan lagi!
